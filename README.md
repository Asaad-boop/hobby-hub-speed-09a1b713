# HobbyShop Admin ERP

High-speed COD ERP system for Bangladesh, built on Lovable.

<!-- sync-test: 2026-04-27 — verifying Lovable ↔ GitHub bidirectional sync -->

## Tech Stack

- TanStack Start (React 19 + Vite 7)
- Tailwind CSS v4
- Lovable Cloud (Supabase)
- TanStack Query

## Key Routes

- `/admin` — Dashboard
- `/admin/orders-workspace` — High-speed COD order processing workspace
- `/admin/web-orders` — Detailed order management
- `/admin/products`, `/admin/inventory`, `/admin/customers`
